# ARROW RUNNER（アロー・ランナー）

## ゲームの内容
本作は2D横スクロール型アクションゲームでプレイヤーは弓を使うキャラクターを操作し、
敵が出現するステージを右方向へ進んでいくゲームとなっています

背景にはピクセル調の森を配置し、ファンタジー世界観を意識した構成になっています
ステージは一本道で敵によってプレイヤーの行動選択が変わるように設計しています


## 操作説明
左右移動：← →キー（またはA / Dキー）

ジャンプ：Spaceキー

攻撃（弓）：Zキー

## アピールポイント

Unityの2D機能を活用し、キャラクター制御・敵挙動・カメラ追従を実装しています

Cinemachineを使用したカメラにより視点制御を行っています

アセットストアの素材を組み合わせ、短期間でも完成度の高い見た目を意識しました

敵のコライダーの範囲を決め、目に当てないと倒せないなど工夫しました

## AI利用で難しかったこと

実装が増えるにつれて、既存スクリプトとの整合性を取るのが難しかった

自分の理解不足により、修正に時間がかかる場面があった

## フィードバック
世界観やピクセルアートの雰囲気は良い

弱点（目など）が分かりづらいため、視覚的に分かる工夫があると良い

ダメージやHPの概念がなく、緊張感が薄い

操作がやや難しく、攻撃キーの配置を見直すと良い

カメラが右に寄っており、進行方向が見えづらい
